## Installation Process

- clone from repo.
- run "composer install".
- copy .env file to .env-example.
- run "php artisan migrate --seed".

## Contributing

- fork the repo.
- clone the repo locally.
- run "git checkout dev"
- create new branch locally.
- work and push to local branch.
- merge to dev
